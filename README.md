# MVVMArch
Sample App for demonstrating Android App Architecture using MVVM + Repository pattern using [Architecture Components](https://developer.android.com/topic/libraries/architecture/) based on:
- [Google's Guide to App Architecture](https://developer.android.com/jetpack/docs/guide)
![Image](https://developer.android.com/topic/libraries/architecture/images/final-architecture.png)
